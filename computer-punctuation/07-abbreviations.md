# 07. 缩写汇总 (Abbreviations)

### AOP
- **中文**：面向切面编程
- **全称**：Aspect-Oriented Programming
- **说明**：将横切关注点（日志、事务）分离的编程范式
- **相关**：OOP、Decorator、Cross-Cutting Concern

### API
- **中文**：应用程序编程接口
- **全称**：Application Programming Interface
- **说明**：软件组件间交互的接口规范
- **相关**：Interface、REST、SDK

### CLI
- **中文**：命令行界面
- **全称**：Command Line Interface
- **说明**：通过文本命令与计算机交互的界面
- **相关**：Shell、Terminal、GUI

### CRUD
- **中文**：增删改查
- **全称**：Create, Read, Update, Delete
- **说明**：数据持久化的四种基本操作
- **相关**：Database、API、REST

### CSS
- **中文**：层叠样式表
- **全称**：Cascading Style Sheets
- **说明**：描述 HTML 文档样式的语言
- **相关**：HTML、DOM、Frontend

### DDD
- **中文**：领域驱动设计
- **全称**：Domain-Driven Design
- **说明**：以业务领域为核心的软件设计方法
- **相关**：Architecture、Microservices

### DOM
- **中文**：文档对象模型
- **全称**：Document Object Model
- **说明**：HTML/XML 文档的编程接口
- **相关**：HTML、JavaScript、Browser

### EOF
- **中文**：文件结束
- **全称**：End Of File
- **说明**：表示数据流结束的标记
- **相关**：Stream、File、Redirection

### ES
- **中文**：ECMAScript
- **全称**：ECMAScript
- **说明**：JavaScript 的标准化规范
- **相关**：JavaScript、TypeScript、Node.js

### FP
- **中文**：函数式编程
- **全称**：Functional Programming
- **说明**：以函数为基本单元的编程范式
- **相关**：OOP、Lambda、Immutable

### GC
- **中文**：垃圾回收
- **全称**：Garbage Collection
- **说明**：自动回收不再使用内存的机制
- **相关**：Memory、Heap、Java

### GDB
- **中文**：GNU 调试器
- **全称**：GNU Debugger
- **说明**：开源的程序调试工具
- **相关**：Debugging、Compiler

### GUI
- **中文**：图形用户界面
- **全称**：Graphical User Interface
- **说明**：图形化的用户交互界面
- **相关**：UI、Frontend、CLI

### HTML
- **中文**：超文本标记语言
- **全称**：HyperText Markup Language
- **说明**：网页内容的标准标记语言
- **相关**：CSS、JavaScript、DOM

### HTTP
- **中文**：超文本传输协议
- **全称**：HyperText Transfer Protocol
- **说明**：Web 通信的基础协议
- **相关**：HTTPS、REST、API

### HTTPS
- **中文**：安全超文本传输协议
- **全称**：HyperText Transfer Protocol Secure
- **说明**：加密的 HTTP 协议
- **相关**：HTTP、SSL、TLS

### I/O
- **中文**：输入/输出
- **全称**：Input/Output
- **说明**：程序与外部设备的数据交换
- **相关**：File、Network、Stream

### IDE
- **中文**：集成开发环境
- **全称**：Integrated Development Environment
- **说明**：编码、调试、构建一体化工具
- **相关**：Editor、Compiler、Debugger

### IPC
- **中文**：进程间通信
- **全称**：Inter-Process Communication
- **说明**：操作系统提供的进程间数据交换机制
- **相关**：Process、Pipe、Socket

### JIT
- **中文**：即时编译
- **全称**：Just-In-Time
- **说明**：运行时将字节码编译为机器码的技术
- **相关**：Compiler、Interpreter、VM

### JS
- **中文**：JavaScript
- **全称**：JavaScript
- **说明**：Web 前端的主流编程语言
- **相关**：TypeScript、ES、Node.js

### JSON
- **中文**：JavaScript 对象表示法
- **全称**：JavaScript Object Notation
- **说明**：轻量级数据交换格式
- **相关**：XML、YAML、API

### JVM
- **中文**：Java 虚拟机
- **全称**：Java Virtual Machine
- **说明**：运行 Java 字节码的虚拟机
- **相关**：Java、Bytecode、GC

### LHS
- **中文**：左侧
- **全称**：Left-Hand Side
- **说明**：赋值运算符的左侧
- **相关**：RHS、Assignment

### LOC
- **中文**：代码行数
- **全称**：Lines of Code
- **说明**：软件规模的度量单位
- **相关**：KLOC、Metric

### MVC
- **中文**：模型-视图-控制器
- **全称**：Model-View-Controller
- **说明**：经典的软件架构模式
- **相关**：MVVM、Architecture、Frontend

### MVVM
- **中文**：模型-视图-视图模型
- **全称**：Model-View-ViewModel
- **说明**：MVC 的变体，常用于前端框架
- **相关**：MVC、Vue、WPF

### NPE
- **中文**：空指针异常
- **全称**：Null Pointer Exception
- **说明**：访问空引用导致的运行时错误
- **相关**：Null Safety、Exception、Bug

### npm
- **中文**：Node 包管理器
- **全称**：Node Package Manager
- **说明**：JavaScript 的包管理工具
- **相关**：Node.js、Yarn、Package

### OOP
- **中文**：面向对象编程
- **全称**：Object-Oriented Programming
- **说明**：以对象为基本单元的编程范式
- **相关**：Class、Inheritance、Polymorphism

### OS
- **中文**：操作系统
- **全称**：Operating System
- **说明**：管理计算机硬件资源的系统软件
- **相关**：Linux、Windows、Kernel

### PID
- **中文**：进程标识符
- **全称**：Process Identifier
- **说明**：操作系统分配的唯一进程编号
- **相关**：Process、Shell、$$、$!

### POSIX
- **中文**：可移植操作系统接口
- **全称**：Portable Operating System Interface
- **说明**：操作系统接口标准
- **相关**：Unix、Linux、Standard

### REPL
- **中文**：读取-求值-输出循环
- **全称**：Read-Eval-Print Loop
- **说明**：交互式编程环境
- **相关**：Interpreter、Shell、Python

### REST
- **中文**：表述性状态转移
- **全称**：Representational State Transfer
- **说明**：基于 HTTP 的架构风格
- **相关**：API、HTTP、Web Service

### RHS
- **中文**：右侧
- **全称**：Right-Hand Side
- **说明**：赋值运算符的右侧
- **相关**：LHS、Assignment

### RPC
- **中文**：远程过程调用
- **全称**：Remote Procedure Call
- **说明**：调用远程服务像调用本地函数
- **相关**：API、gRPC、Distributed System

### RTTI
- **中文**：运行时类型信息
- **全称**：Run-Time Type Information
- **说明**：运行时获取对象类型的机制
- **相关**：Type、Reflection、C++

### SaaS
- **中文**：软件即服务
- **全称**：Software as a Service
- **说明**：通过云提供软件应用
- **相关**：PaaS、IaaS、Cloud

### SDK
- **中文**：软件开发工具包
- **全称**：Software Development Kit
- **说明**：开发特定平台应用的工具集
- **相关**：API、Library、Framework

### SQL
- **中文**：结构化查询语言
- **全称**：Structured Query Language
- **说明**：关系数据库的标准查询语言
- **相关**：Database、RDBMS、ORM

### SSH
- **中文**：安全外壳协议
- **全称**：Secure Shell
- **说明**：加密的远程登录协议
- **相关**：Telnet、Remote、Security

### SSL
- **中文**：安全套接层
- **全称**：Secure Sockets Layer
- **说明**：已弃用的加密协议，现由 TLS 替代
- **相关**：TLS、HTTPS、Security

### stdin
- **中文**：标准输入
- **全称**：Standard Input
- **说明**：程序默认的输入流（文件描述符 0）
- **相关**：stdout、stderr、I/O、Redirection

### stdout
- **中文**：标准输出
- **全称**：Standard Output
- **说明**：程序默认的输出流（文件描述符 1）
- **相关**：stdin、stderr、I/O、Redirection

### stderr
- **中文**：标准错误
- **全称**：Standard Error
- **说明**：程序默认的错误输出流（文件描述符 2）
- **相关**：stdin、stdout、I/O、Redirection

### TDD
- **中文**：测试驱动开发
- **全称**：Test-Driven Development
- **说明**：先写测试再写代码的开发方法
- **相关**：Unit Testing、Agile、XP

### TLS
- **中文**：传输层安全
- **全称**：Transport Layer Security
- **说明**：SSL 的继任者，加密传输协议
- **相关**：SSL、HTTPS、Security

### TTY
- **中文**：电传打字机终端
- **全称**：Teletypewriter
- **说明**：终端设备的统称
- **相关**：Terminal、Shell、CLI

### TS
- **中文**：TypeScript
- **全称**：TypeScript
- **说明**：JavaScript 的超集，添加了类型系统
- **相关**：JavaScript、ES、Type

### UI
- **中文**：用户界面
- **全称**：User Interface
- **说明**：人机交互的界面
- **相关**：UX、GUI、Frontend

### URI
- **中文**：统一资源标识符
- **全称**：Uniform Resource Identifier
- **说明**：标识资源的字符串
- **相关**：URL、URN、Web

### URL
- **中文**：统一资源定位符
- **全称**：Uniform Resource Locator
- **说明**：定位资源的地址
- **相关**：URI、HTTP、Web

### UTF-8
- **中文**：Unicode 转换格式-8
- **全称**：Unicode Transformation Format-8
- **说明**：变长字符编码，兼容 ASCII
- **相关**：Unicode、Encoding、ASCII

### UX
- **中文**：用户体验
- **全称**：User Experience
- **说明**：用户使用产品的整体感受
- **相关**：UI、Design、Usability

### VM
- **中文**：虚拟机
- **全称**：Virtual Machine
- **说明**：模拟硬件执行代码的软件环境
- **相关**：JVM、Docker、Virtualization

### XML
- **中文**：可扩展标记语言
- **全称**：eXtensible Markup Language
- **说明**：标记数据的可扩展格式
- **相关**：HTML、JSON、YAML

### YAML
- **中文**：YAML 不是一种标记语言
- **全称**：YAML Ain't Markup Language
- **说明**：人类可读的数据序列化格式
- **相关**：JSON、XML、Config

### YARN
- **中文**：又一个资源协调者
- **全称**：Yet Another Resource Negotiator
- **说明**：Hadoop 资源管理器，也指 JS 包管理器
- **相关**：npm、Package Manager

### ZSH
- **中文**：Z Shell
- **全称**：Z Shell
- **说明**：功能强大的 Unix Shell
- **相关**：Bash、Shell、Oh-My-Zsh
