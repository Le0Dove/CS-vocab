## 核心概念

计算机网络是计算机科学的核心领域，核心内容包括：

- **网络基础**：带宽、延迟、吞吐量、OSI 模型、TCP/IP 模型
- **数据链路层**：帧、MAC 地址、交换、VLAN、STP
- **网络层**：IP 地址、子网划分、路由（OSPF、BGP）
- **传输层**：TCP 三次握手/四次挥手、流量控制、拥塞控制、UDP
- **应用层**：HTTP/DNS/CDN/邮件协议

词汇按字母顺序排列，方便检索。
