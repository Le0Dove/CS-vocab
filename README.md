# CS-Vocab

## 使用

```bash
git clone https://github.com/yourusername/cs-vocab.git
```

- **在线浏览**：直接点击上方科目链接查看 Markdown 文件
- **本地搜索**：用编辑器全局搜索快速定位词汇
- **考研/面试**：系统复习操作系统英文术语
- **论文阅读**：查阅不熟悉的概念和缩写
