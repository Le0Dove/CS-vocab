## 核心概念

数据结构与算法是计算机科学的基石，核心内容包括：

- **数据结构**：线性结构、树结构、图结构、哈希结构
- **算法分析**：时间复杂度、空间复杂度、渐进分析
- **经典算法**：排序、搜索、字符串匹配、图算法
- **设计策略**：分治、动态规划、贪心、回溯、分支限界
- **算法理论**：P vs NP、NP-Complete、可计算性

词汇按字母顺序排列，方便检索。
