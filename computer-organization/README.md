## 核心概念

计算机组成原理研究计算机硬件系统的结构与工作原理：

- **数据表示**：二进制、补码、浮点数、字符编码
- **CPU 设计**：数据通路、控制器、流水线、指令级并行
- **存储层次**：缓存、主存、虚拟内存
- **I/O 系统**：总线、中断、DMA

词汇按字母顺序排列，方便检索。
